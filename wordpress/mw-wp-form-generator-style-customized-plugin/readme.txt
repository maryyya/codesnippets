== Description ==

Style for MW WP Form Generator. This is for doginsoken.
This was originally created by PPC Group

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the style in MW WP Form creation page.
